The Agent Skill Library is the planned reusable-knowledge layer for businesses
that need a capable specialist, not a freshly installed general-purpose agent.
Each skill pack would combine domain expertise, operating instructions,
integration contracts, authority limits, evaluation cases, and the evidence a
successful run must return.

## The job

A skill pack is intended to shorten the distance between “the agent is online”
and “the agent can perform this job reliably.” Example families could cover
local-search audits, real-estate research, revenue qualification, customer
support, or financial telemetry. A pack may include compatible agent skill
documents, templates, reference material, test fixtures, and configuration
guidance without embedding customer secrets.

## The boundary

Skills are inspectable and versioned so a buyer can see what changed and which
evaluation supports the release. They do not grant ambient authority. Runtime
credentials, data access, money movement, publishing, and other consequential
actions stay governed by the deployment that installs the skill.

## Availability

This library is planned and not yet live or generally available. No catalogue
entry claims that a vertical pack has passed production evaluation, produced a
customer result, or is safe for unsupervised use. Those claims require real
evaluation and runtime receipts. Read the [managed agent infrastructure
operating thesis](https://github.com/fongryan/portfolio/blob/main/docs/managed-agent-infrastructure.md).
